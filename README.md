# Parmod Yadav — Portfolio 👨‍💻

Welcome to my personal portfolio website! This project showcases my skills, projects, and passion as a **MERN Stack Developer** and **Automobile Enthusiast**. Built with **React + Vite**, it’s a fast, modern, and responsive web app to introduce who I am, what I build, and what drives me.

## 🚀 Live Demo

[View Portfolio Live](https://parmodkumar28.netlify.app/)

## 🛠 Tech Stack

- **Frontend**: React, Vite, Tailwind CSS
- **Animations & Icons**: Framer Motion, SVG assets
- **Tooling**: ESLint, Prettier

## 📂 Project Structure
 public/ → Static assets
 src/ → Application source code
 ├── assets/ → Images & icons
 ├── components/→ Reusable UI components
 ├── App.jsx → Main application logic
 └── main.jsx → React entry point


## 💡 Features

- Sleek and responsive UI
- Projects gallery with visuals
- About section with personal bio
- Contact section with social links
- Smooth animations and transitions
